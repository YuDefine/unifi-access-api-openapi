# UniFi Access API OpenAPI Project

This project provides an unofficial OpenAPI 3.1.0 specification for the UniFi Access API.

## Key Files

- `openapi.yaml` — The OpenAPI spec (source of truth for all endpoints and schemas)
- `docs/unifi-access-api.md` — Cleaned Markdown from the original PDF
- `docs/USAGE.md` / `docs/USAGE.zh-TW.md` — Usage guides

## For AI: Working with This API

When asked to write code that calls UniFi Access API, read `.claude/skills/unifi-access-api/SKILL.md` first — it contains the full endpoint quick reference, authentication details, response format, and code examples.

For detailed parameter/schema info, read the relevant section of `openapi.yaml` directly.
