## Getting Started

1. **Create an API Token** — Go to your UniFi Console → Access → Settings → General → Advanced → API Token
2. **Optionally enter your host and API key** above so code examples are ready to use
3. **Browse endpoints** from the sidebar or use the search bar

::: tip
The API host is typically `https://<your-console-ip>:12445`. All requests require HTTPS.
:::
