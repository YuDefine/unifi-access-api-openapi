# API 快速查找

依分類瀏覽所有 UniFi Access API 端點。從側邊欄選擇任一操作，查看其詳細資訊、參數與回應結構。

## 設定您的環境（選填）

輸入您的 UniFi Access 主機位址與 API Key，讓程式碼範例可以直接複製使用。

::: info 隱私說明
此為**選填**的便利功能，僅供測試使用。所有數值僅儲存於您瀏覽器的 localStorage 中，**不會進行任何資料傳輸或伺服器端保存**。若您有疑慮，可至 [GitHub fork 本專案](https://github.com/YuDefine/unifi-access-api-openapi)後在本機使用，或自行複製範例程式碼到本機執行測試。
:::

## 快速開始

1. **建立 API Token** — 前往 UniFi Console → Access → Settings → General → Advanced → API Token
2. **可選擇在上方輸入主機位址與 API Key**，讓程式碼範例可直接使用
3. **從側邊欄瀏覽端點**或使用搜尋功能

::: tip
API 主機位址通常為 `https://<your-console-ip>:12445`，所有請求皆需使用 HTTPS。
:::
