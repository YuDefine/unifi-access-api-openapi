# 變更日誌

## v4.0.10
- 支援在 Intercom 和 Reader Pro 上觸發門鈴。(#8.4)

## v3.3.21
- 支援遠端開門 API，可自訂 actor_id、actor_name 及 Webhook 輸入中的 extra 欄位。若省略，系統將預設使用 Token 名稱作為日誌中的操作者。(#7.9)

## v3.3.10
- 支援指派與取消指派使用者車牌。(#3.28-#3.29)
- 支援更新使用者大頭照。(#3.30)
- 支援指派與取消指派訪客車牌。(#4.13-#4.14)
- 支援指派與取消指派訪客 QR Code 憑證。(#4.11-#4.12)
- 支援下載使用者 QR Code 圖片。(#6.18)
- 支援匯入第三方 NFC 卡片 ID 至系統。(#6.19)
- 支援取得與更新特定裝置的存取方式設定。(#8.2-#8.3)
- 支援額外的 Webhook 事件。(#11.2)

## V3.2.20
- 支援 Touch Pass API (#3.25-#3.27, #6.11-#6.17)
- 若使用者已被指派 Touch Pass，回應中將包含 Touch Pass 詳細資訊。(#3.4-#3.5)

## V3.1.30
- 支援 Webhook 自訂標頭設定與轉發。(#11.3)
- 支援緊急事件 Webhook 通知。(#11.2)
- 支援 EAH8 與 UA-Hub-Gate 的 DPS 狀態變更 Webhook 通知。(#11.2)
- 支援使用者刪除 API。(#3.23)
- 支援使用者搜尋 API。(#3.24)
- 支援 NFC 卡片更新 API。(#6.10)
- 支援 EAH8、UA-Hub-Door-Mini 及 UA-Ultra 的臨時解鎖。需要最新的裝置版本。(#7.10-#7.11)

## V2.2.6
- 支援使用者群組與使用者群組政策。(#3.11-#3.22,#10.5-#10.6)
- 支援 Access Ultra (UA-Ultra) 的遠端開門。(#7.9)

## V2.2.10
- 支援 Webhook API。(#11.2-#11.7)
- 支援 API 伺服器憑證。(#12.1-#12.2)
